    iParcel: A Smart Parcel Management System

    The iParcel system was developed to address inefficiencies in parcel management and improve security for Mahallah Zubair residents. It streamlines parcel registration, scheduling, and collection processes, offering an intuitive interface for both users and administrators.

    Key Features

    • User Dashboard: Track parcels, schedule pickups (standard or urgent), and upload payment proofs.
    • Admin Dashboard: Manage parcels, monitor user accounts, and generate automated reports.
    • Scheduling: Flexible pickup scheduling with standard and urgent options, reducing congestion during peak hours.
    • Reporting: Generate detailed summaries with visual graphs for parcel operations and revenue tracking.
